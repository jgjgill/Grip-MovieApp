# ✨Grip-MovieApp

# 📝소개

wanted_pre_onboarding에서 진행한 기업 과제로 그립컴퍼니 개발 사전 과제 구현 내용입니다.

영화를 검색하고 즐겨찾기로 등록할 수 있는 React 앱을 구현했습니다.

# ⚡️구현 내용

- API를 통한 영화 검색 기능

- 영화 즐겨찾기 기능
